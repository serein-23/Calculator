# Qt Calculator

这是一个使用 Qt 框架实现的简单计算器应用程序。它支持基本的算术运算，包括加法、减法、乘法和除法，以及括号来改变运算顺序。

## 功能

- 支持加法（+）、减法（-）、乘法（*）、除法（/）运算。
- 支持使用括号来设置运算优先级。
- 提供清除（C）和删除（<—）功能，方便用户修改表达式。
- 结果显示在输入框上方。
